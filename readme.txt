== Installation ==

1. Upload the entire `scratchblocks-for-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. use shortcode [scratchblocks element=pre]Scratch Block Syntax[/scratchblocks]
http://wiki.scratch.mit.edu/wiki/Block_Plugin#Basic_Syntax
